CSS - отдельный язык пердназначенный для определения  стилевых сайтов
HTML элементов на странице по заданному селектору

Синтаксис языка CSS
< селектор>{

<свойство>:<значение>;
<свойство>:<значение>;

}

* селектор - конструкция при помощи которой будет производиться посик элемента по заданному критерию

* свойство - название одного из СSS свойств заранее определенного в стандарте языка

* значение - значение для определенного свойства

Типы CSS селекторов:
1 - По названию HTML элемента - поиск производится по названию HTML тега без доплнительных обозначений
div{} - выбрать все элменты div на странице
a {} - выбрать все элементы а на странице
div a {} - выбрать все элементы а одним из родителей которых является div
div > a {} - выбрать все элементы а непосредственным родителем которых является div

2 - По классификатору HTML элемента - поиск производится по названию класса заданного HTML элемента при помощи аттрибта
class. Один класс может содержать в себе заданных стилевых свойств и применятся к нескольким HTML элементам
К одному HTML элементу может примеяться несколько классов

.box {} - выбрать все элементы с классом box.
.box.yellow {} - выбрать все элементы у которых заданы 2 класса box и yellow
.box .link {} - выбрать все элементы с классом link одним из родителей  которых явлается элемент с классом box
.box > .link {} - выбрать все элементы с классом link непосредственным родителем которых явлется элемента с калссом
box.


3 - По идентификатору HTML элемента - поиск проводится по уникальному названию элемента заданного при помощи
аттрибута id. Нельзя задать один и тот же id для двух разных HTML элементов.
user_form {} - выбрать один единственный элемент с идентификатором user-form.

Основное отличие идентификатора от класса заключается в то что класс можно применить к нескольким элементам
на странице,а идентификатор только к одному.

Любой HTML элемент может состоять из следующих слоев:

1) Внутренняя область (content) - область внутри которой будет находится все элементы созданные внутри исходного
2) Область внутренних отступов (padding) - расстояние на которого внутренняя область будет отодвинута от границ
элемента
3) Область внутренних границ (border) - обводка элемента

  'box-sizing'  - свойство определяющее поведение  ширины и высоты для исходного элемента

 * content-box -  размеры будут заданы исключительно для внутренней области элемента
 * border-box - размеры будут заданы как окончательные значения для всего элемента

 .box {box-sizing : content-box}
  Итоговый размер -> content(initial size) + padding + border


  .box{box-sizing : border-box}
  Итоговый размер -> border (initial size) - border - padding = content.

  В CSS существует 4 базовых направления для работы со свойствами :

 1 top - верх
 2 left - лево
 3 right - право
 4 bottom - низ

 padding - внутренний отступ со всех направлений
 padding left - внутренний отсут с левого направления

  Свойства padding и margin позволяют производить запись через разное количество параметров

    padding: 25px - отступ в 25px со всех направлений
    padding: 15px 25px - отступ 15 рх   сверху и снизу + 25 рх слева и справа

    'padding : 15px 25px 10px 5px ' - по часавой стрелке

    У margin есть возможность задать в качестве значениа auto, в таком случае элемент будет смещен на противоположную
    сторону от исходного направления

    margin-left auto: - элемент сместиться в правую сторону
    margin-right auto: - элемент сместиться в левую сторону

    border:  1 2 3 - свойство для форммирования внутренней границы элемента
    1 - толщина линии
    2 - тип линии (solid,dotted,dashed - так далее)w
    3 - цвет линии

    border-radius - свойство для скругления углов по заданному по окружности заданным радиусом

    border-top-left-radius - скругление для верхнего левого  угла
    border-top-right-radius - скругление для верхнего правого  угла
    border-bottom-left-radius - скругление для нижнего левого  угла
    border-bottom-right-radius - скругление для нижнего правого  угла

    box-shadow - для формирования внут или наруж тени
    1 - Смещение тени по горизонтали
    2 - По вертикали
    3 - Расстояние размытия тени
    4 - цвет тени

    font-size - размер текста на странице.По дефолту 16

    rem- относительная еденица измерения работающая относительно базового
    размера текста

    1rem - 100% от базового текста (16рх)

    1.5rem - 150% (24рх)


     font-weight - толщину шрифта Диапазон от 100 до 900 с интервалом 100,по дефолту 400

     text-decoration 1 2 3 4
      * underline - линия под текстом
      * overline - линия над текстом
      * line-through - зачеркнутый текст
      * none - никакой декорации
