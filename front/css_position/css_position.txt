Позиционирование - группа свойств регулирующих расположение элемента на странице относительно
других элементов. Расположение элемента в позиционировании регулируется помощи 4-х свойств:top,left,right,bottom.

Свойства регулировки позиции могут работать по разному в зависимости от используемого позиционирования.

Тип позиционирования для элемента определяется при помощи свойства position.

Возможные тип поизиц-я:

1) static (статическое) - тип позиционрирования по умолчанию.При использовании элемент будет находиться в своей позиции
по умолчанию и при этом свойства регулировки позиции работать не будут.


2) relative (относительное) - тип позиционирования при котором элемет изначально будет находиться в своей позиции
по умолчанию , а с регулировки позиции будут работать относительно границ начального местоположения элемента


3) absolute (абсолютное) - тип позициониирования при котором элемент полностью вырывается из своей
начальной структуры и привязывается к первому не статичному родительскому элеементу либо если такого элемента нет
то к окну браузера.Свойства регулировки позициий будут работать относительно того элемента к котророму привязан
абсолютный.
