Flex - группа CSS свойств предназначенная для регу  лировки позиции элементов относительно
flex-контейнер


Flex-контейнер - HTML элемент у которго задано свойство display со значением flex либо inline-flex
У flex-контейнера вводятся две направлении линии называемые онсновной и поперечной осью

Основная ось - направляющая линия по которой распологатьася непосредственно вложенные во flex-контейнер
элементы

Поперечная ось - дополнительная направляющая линия пересекающая основную

Возможные направления основной оси
 row - слева на право
 row-reverse - справа на лево

 column - сверху вниз
 column-reverse - снизу вверх


 Направлющая поперечной оси определяется автоматический в зависимости от направления
 основной оси Поперечная ось не может быть обратной (reverse)
 Направляющая поперечной оси по отношению к основной

 Если основная row то поперечная column
 Если основная row-reverse то поперечная column
 Если основная column-reverse  то поперечная row

 flex-direction - определяет направление основной оси


 justify-content - регулирует расположение элементов flex-контейнера по основной оси

 flex-start - в начале оси
 flex-end - в конце оси
 center - по центру оси
 space-between - элементы будут растянуты по всему пространству основной оси с равными интервалами между собой

align-items - регулирует расположение элементов flex-контейнера по поперечной оси

flex-start - в начале оси
flex-end - в конце оси
center- по центру оси
stretch - растягивает элемент на все пространство поперечной оси при условий у элемента не задан размер
