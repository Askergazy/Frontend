Псевдокласыы - это доп-й селектор позвволяющий выбрать элемент в
определнном состоянии либо последовательности. Псевдоклассы начинается с
символа :(псевдокласс старого образца) либо :: (псевдокласс нового образца)

:hover - отлавливает наведение на элемент

:active - отлавливает нажатие на элемент левой мыши

:focus -  отлавливает фокусировку на элементе (актуально для текстоввых полей и
различных элементов формы для пользоввательского ввода).

'.box:hover {}' - выбрат с элементом бокс вв момент наведения на него.

 .box:hover .sub {} - выбрать элемент с классом sub родителем котрого
  выступает элемент с класслм box в момент наведения по нему.

 'transition' - група свойств позволяющих задать плавное петикание от начального значения
 свойства к конечному в течение заданного промежутка времени.

 transition-property - определяет css свойства (ЧЕРЕЗ ЗАПЯТУЮ) к которым должент быть применен плавный
 переход по умолчанию в качестве значения используется all (все свойства)

  transition-duration - определяет продолжителность плавного переходаа от начальных значений свойства к конечным
  в секундах
